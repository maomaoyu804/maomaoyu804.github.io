[数论]bzoj3884 上帝与集合的正确用法

发表于 2020-04-28 更新于 2020-05-16 分类于 ACM ， 数论

扩展欧拉定理

题意：

输入及输出

Input
接下来T行，每行一个正整数p，代表你需要取模的值

Output
T行，每行一个正整数，为答案对p取模后的值

Sample Input
3
2
3
6

Sample Output
0
1
4

HINT
对于100%的数据，T<=1000,p<=10^7

题解

令$m=2^{2^{2^{…}}}$，即求$m%p$。
考虑到m和p不一定互质，易想到扩展欧拉函数：

$$
a^b  mod  p=
\begin{cases}
a^{b\%\phi(p) } mod  p &gcd(a,p)=1\\
a^b  mod  p &gcd(a,p)\neq 1,b\leq \phi(p)\\
a^{b\%\phi(p)+\phi(p)}  mod  p &gcd(a,p)\neq 1,b\geq\phi(p)
\end{cases}
$$

因此，令$f(p)=m\%p$，则有
$$
\begin{aligned}
f(p)&=m\%p\\
&=2^m\%p\\
&=2^{m\%\phi(p)+\phi(p)}\%p\\
&=2^{f(\phi(p))+\phi(p)}\%p\\
\end{aligned}
$$
于是可以递归求解，复杂度约为$O(\sqrt p * \log p)$
