这篇博客主要记录github的搭建过程，以及使用说明

2022.2.17

  1. 博客大搬家，具体见这里。
  2. 增加“关于“
    hexo new page "about"
    在/source下会新生成一个新的文件夹about，该文件夹下有index.md文件。
    themes/next/_configy.yml 中menu中去掉about前注释。
  3. 把一些之前写在notion的东西搬过来了

2020.5.16

  1. 新增友链
    添加方法：I:\Workspaces\hexo\themes\next\_config.yml中搜索Blog rolls，在links下面添加即可。
  2. 对文章重新进行分类

2020.5.15

  1. 新增分类功能
    git bash中输入
    $ hexo new page categories
    成功后提示
    INFO Created: ~I:/Workspaces/hexo/source/source/categories/index.md
    根据上面的路径，找到index.md，最上面添加type: "categories"
    在文章配置中添加如下配置，即可实现分类的嵌套。 参考：https://www.jianshu.com/p/e17711e44e00
  2. 目前待解决问题：
    插入连续行间代码块会自动合并、及附近缩进问题（目前加tab可以强行解决）
    按嵌套逻辑对文章重新分类

2020.4.28

  1. 公式中转义符号为\\，例如：\\ $\%$为$\%$、\\\\为换行符
    文本中转义符号只需要\，例如\%为\%
  2. 设置摘要
    <!--more-->

2020.4.18

  1. 安装latex插件，能正常显示公式
    需要在.md文件开头加上mathjax: true
    .md文件路径为\source\_posts
    参考：https://www.dazhuanlan.com/2020/01/06/5e12e91f4f2e0/
    注意公式内斜体字可能报错，* 后面记得加空格

  2. 安装hexo-asset-image插件，可以插入图片
    参考：https://blog.csdn.net/qq904305159/article/details/104162802
    图片存放路径： \source\images
    链接方法：![我是图片](/images/logo.png)
    参考：https://blog.csdn.net/nineya_com/article/details/103321401
